TabNap Privacy Policy

Last updated: July 19, 2026

TabNap is a Chrome extension that suspends inactive tabs to free RAM. All tab management runs locally in your browser.

Data we access

  • Tab metadata (URL, active state, pinned, audible) to decide which tabs to suspend
  • Extension preferences stored in chrome.storage.sync (whitelist, auto-suspend timer)

Data we do not collect

  • No account required
  • No browsing history sent to external servers
  • No sale or transfer of user data to third parties
  • No remote code — all scripts are bundled in the extension package

Permissions

TabNap uses the tabs permission to suspend and restore tabs when you request it or when auto-suspend runs. It does not read page content.
