# TabNap

Auto-suspend inactive Chrome tabs to **free RAM** and speed up your browser.

## Features
- One-click "Nap all other tabs"
- Wake all sleeping tabs
- Auto-suspend after 30 min (configurable)
- Whitelist Gmail, Docs, Meet by default
- Never naps pinned or playing-audio tabs

## Install (dev)
1. `chrome://extensions` → Developer mode
2. Load unpacked → this folder

## Target users
Anyone with 20+ tabs open (students, devs, researchers) — **100k+ potential**

## Monetization (v1.1 Pro)
- Per-site auto rules
- Suspend across all windows
- RAM saved stats
- ₹149 lifetime
